📰 News Website with Admin Panel and OAuth Authentication (Spring Boot & React)

This project aims to build a News Website featuring a Home Page, Admin Panel for managing news, and OAuth-based authentication for both users and administrators.

📌 Project Overview

🏠 Home Page

  • Display news articles, trending stories, and category filters.
  • Fully responsive design for desktop and mobile.
  • Pagination and search functionality.

🔐 Admin Panel

  • Admins can add, edit, or delete news articles.
  • Role-based access control.
  • Dashboard with analytics.

⚙️ Technology Stack

🌐 Frontend (React + Vite)

  • React.js - Dynamic UI.
  • Tailwind CSS - Modern styling.
  • Redux Toolkit - State management.

🔧 Backend (Spring Boot + Security)

  • Spring Boot - Secure API.
  • JWT & OAuth - Authentication.
  • MySQL/PostgreSQL - Database.

🚀 Project Roadmap

  • Phase 1: Backend & Authentication Setup.
  • Phase 2: API Development & Admin Panel.
  • Phase 3: Frontend Integration.
  • Phase 4: Enhancements & Performance Optimization.
  • Phase 5: Testing & Deployment.

🔮 Future Enhancements

  • Real-time news updates with WebSockets.
  • Push notifications for breaking news.
  • AI-powered content moderation.
